# 🚀 TaskFlow // Professional Todo Manager

TaskFlow is a modern, responsive, and high-performance task management application built with pure Vanilla JavaScript. It features a sleek glassmorphic UI, dark mode support, and persistence using Local Storage.

![https://todo-app-topaz-gamma-21.vercel.app/]

## ✨ Features

- **Modern UI/UX**: Clean, professional design with smooth CSS transitions.
- **Dark Mode**: Fully implemented system-aware and manual theme switching.
- **Full CRUD**: Add, Read, Update, and Delete tasks seamlessly.
- **Smart Filtering**: Filter by All, Pending, or Completed tasks.
- **Local Storage**: Data persists even after page refreshes.
- **Responsive**: Mobile-first design that looks great on all screens.
- **Accessibility**: Semantic HTML and keyboard-friendly interactions.
- **Empty State**: Beautifully designed empty state message.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom variables, Flexbox, Grid, and Animations.
- **JavaScript (ES6+)**: Functional programming, DOM manipulation, and Local Storage.
- **FontAwesome**: For professional iconography.
